Aulas da autoescola

Tipos de dados coletados

Ao instalar o aplicativo Aulas da autoescola é gerado, na API Rest do aplicativo, um token de identificação para o usuário. O token é gerado a partir de um algoritmo presente na API e NÃO utiliza identificadores do celular do usuário como o IMEI, ou emails por exemplo. Tal token fica armazenado no celular do usuário e na API Rest. O token gerado e a API tem como finalidade permitir a comunicação, via aplicativo, entre desenvolvedor e usuário. Quando o usuário desinstala o aplicativo, o token é excluído do celular e nunca mais é utilizado.

Período de conservação

O token fica guardado no celular do usuário enquanto o aplicativo estiver instalado nele. Ao desinstalar o aplicativo o token é excluído do celular e nunca mais poderá ser utilizado pelo usuário ou pelo desenvolvedor para estabelecer uma comunicação.

Exibição de publicidade

O aplicativo contém publicidade do Google Admob, sendo apresentada nas telas inicias de cada categoria e na tela de solicitação de novos despertadores.

Contatar o usuário

Dentro do aplicativo o usuário pode se comunicar com o desenvolvedor, e vice-versa, por meio de mensagens. As mensagens podem ser enviadas pelos usuários dentro do próprio aplicativo. As respostas enviadas pelo desenvolvedor serão exibidas também dentro do aplicativo.

Mudanças nesta política de privacidade

O desenvolvedor reserva a si próprio o direito de fazer alterações nesta política de privacidade a qualquer momento. Se o Usuário não concordar com qualquer das alterações da política de privacidade, deverá cessar o uso deste aplicativo.
